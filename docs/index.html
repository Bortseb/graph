Lightweight Property Graph as Javascript Module

Example

Create an empty graph


      import {Graph} from 'https://wardcunningham.github.io/graph/graph.js'
      const graph = new Graph()

Add nodes to the graph, remember node ids.


      const repo = graph.addNode('Repo',{name:'Graph',url:'https://github.com/WardCunningham/graph/'})
      const docs = graph.addNode('Docs',{name:'Graph',url:'https://wardcunningham.github.io/graph/'})

Add relation to the graph, connect remembered nodes by id.


      graph.addRel('Hosts',repo,docs,{})

Convert graph to JSON formatted and indented text.


      const text = graph.stringify(null,2)
